# Zere Fashion House - Website Replica

A complete replica of the Zere Fashion House website built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Product Showcase**: New Arrivals and Spotlight sections with product cards
- **Shopping Cart**: Add/remove items, update quantities, view cart total
- **User Authentication**: Login modal (UI only)
- **Search Functionality**: Search modal with trending categories
- **Newsletter Signup**: Email subscription form
- **Cookie Consent**: Cookie banner with privacy policy link
- **Smooth Animations**: Hover effects and transitions throughout

## Files Structure

```
fashion/
├── index.html      # Main HTML structure
├── styles.css      # All styling and responsive design
├── script.js       # JavaScript functionality
└── README.md       # This file
```

## How to Use

1. Open `index.html` in your web browser
2. No build process or server required - works directly in the browser

## Features Implemented

### Header
- Logo and navigation menu
- Search, user account, and shopping cart icons
- Trending categories banner
- Wishlist counter
- Mobile-responsive menu

### Product Sections
- New Arrivals section
- Spotlight section

### Modals
- Login modal
- Shopping cart modal

### Footer
- Contact information
- Help links
- Delivery & Returns information

### Interactive Features
- Add products to cart
- Update cart quantities
- Remove items from cart
- Newsletter subscription
- Cookie consent banner

## Browser Compatibility

Works on all modern browsers:
- Chrome
- Firefox
- Safari
- Edge

## Notes

- Product images use placeholder images (via.placeholder.com)
- Cart data is stored in browser localStorage
- Login functionality is UI-only (no backend)
- Search functionality is basic (can be enhanced)

## Customization

You can easily customize:
- Colors in `styles.css` (CSS variables at the top)
- Product data in `index.html`
- Cart functionality in `script.js`
- Add more products or sections
